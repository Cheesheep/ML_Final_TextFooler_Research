# 数据目录说明

## 课程数据

- `TextFooler攻击后新的欺诈通话数据.xlsx`

该文件包含 2677 组欺诈通话原文及其 TextFooler 攻击版本，主要用于：

- 监督训练中的正类来源
- TextFooler 鲁棒性评测

## 公开补充数据

- `raw/`：存放下载的 LCCC base 原始压缩文件
- `processed/fraud_benchmark/`：存放最终构建出的实验数据包

## 为什么需要公开补充数据

课程数据只包含欺诈正类及其攻击版本，不包含正常对话负类，因此无法单独完成“欺诈 vs 正常”的监督学习训练。为满足课程实验要求，本文补充引入 LCCC 正常对话子集作为负类，并在论文中明确说明这一数据构造方式。
